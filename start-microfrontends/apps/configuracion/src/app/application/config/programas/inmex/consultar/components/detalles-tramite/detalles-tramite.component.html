Gestión de fracciones arancelarias

  1. Agregar: En esta sección se agregará cada elemento de forma individual y realizando sus validaciones correspondientes según sea el caso.
  2. Eliminar: En esta sección se eliminarán los elementos ya sea de forma individual o por conjuntos por medio de filtros para su mejor visualización, además se cuenta con un botón dónde se podrán eliminar todos los elementos.
  3. Consultar: En esta sección se muestran los elementos cargados ya sea de forma manual o masiva (por archivo) por medio de filtros para una mejor visualización, además se podra descargar un archivo donde se muestran todos los elementos cargados.
  4. Cargar archivo: En esta sección se cargará un archivo con el formato predeterminado, en caso de ser exitoso los resultados se podrán observar en la sección de consulta, si el archivo contiene errores se muestra un botón donde se podrá descargar un archivo indicando la línea y una descripcion del error.
